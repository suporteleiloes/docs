# Ficha do colaborador

A ficha do colaborador reúne, em uma única tela, todos os dados cadastrais de uma pessoa e o seu histórico dentro da leiloeira: os eventos registrados (faltas, férias, advertências etc.) e os lançamentos financeiros (salários, bônus, descontos). É a visão completa de cada colaborador.

## Como acessar

**ERP** → **RH** → **Colaboradores** → clique sobre a linha de um colaborador.

![Ficha do colaborador](/img/manual/erp/rh-colaboradores-id.png)

## O que você vê nesta tela

A tela tem duas partes: o **resumo** do colaborador no topo e, abaixo, o **histórico** organizado em abas.

### Resumo

No cabeçalho aparecem o nome, a situação (Ativo, Inativo, Afastado, Férias, Desligado), o cargo e o departamento. Em seguida, um quadro com os dados principais:

| Campo | O que é |
|---|---|
| CPF | Documento da pessoa |
| Matrícula | Matrícula interna |
| E-mail | E-mail de contato |
| Telefone | Telefone de contato |
| Vínculo | CLT, PJ, Estágio, Temporário ou Autônomo |
| Admissão | Data de admissão |
| Nascimento | Data de nascimento |
| Salário | Remuneração base |

Quando houver, as **Observações** internas aparecem logo abaixo do quadro.

### Histórico (abas)

| Aba | O que mostra |
|---|---|
| Eventos | Eventos registrados para a pessoa: Tipo, Data, Descrição, Valor e Status |
| Financeiro | Lançamentos financeiros: Tipo, Competência, Valor, Vencimento e Status (valores de desconto aparecem em vermelho com sinal de menos) |

## O que dá pra fazer aqui

### Editar os dados do colaborador

1. Clique em **Editar** no topo da tela.
2. Ajuste os campos no formulário que abre.
3. Clique em **Salvar**. A ficha é atualizada na hora.

### Alternar entre Eventos e Financeiro

Clique nas abas **Eventos** ou **Financeiro** (cada uma mostra, entre parênteses, a quantidade de registros) para ver o histórico correspondente.

### Voltar para a lista

Clique em **Voltar** para retornar à lista de colaboradores.

## Dicas e observações

- O histórico desta tela é apenas para consulta. Para criar ou alterar um evento, use a tela de [Eventos](./rh-eventos.md); para lançamentos financeiros, use o [Financeiro de RH](./rh-financeiro.md).
- O histórico (abas Eventos e Financeiro) traz **todos** os registros do colaborador, do mais recente para o mais antigo — não é paginado como as telas de lista.
- Os campos que aparecem como "—" significam que não há informação cadastrada para aquele item.
- A visualização do histórico depende da permissão de **ver relatórios de RH**. Quem só tem acesso ao cadastro de colaboradores enxerga o resumo, mas pode não ver as abas de histórico.
