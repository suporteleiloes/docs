# Cargos

A tela de Cargos é onde você define os cargos existentes na sua leiloeira (por exemplo, Leiloeiro, Auxiliar Administrativo, Avaliador), com nível, departamento e salário-base. Esses cargos ficam disponíveis depois para vincular a cada colaborador.

## Como acessar

**ERP** → **RH** → **Cargos**.

![Cargos](/img/manual/erp/rh-cargos.png)

## O que você vê nesta tela

Um campo de busca no topo e a tabela com todos os cargos cadastrados.

### Colunas da tabela

| Coluna | O que é |
|---|---|
| Cargo | Nome do cargo |
| Nível | Júnior, Pleno, Sênior ou Liderança |
| Departamento | Departamento ao qual o cargo está ligado |
| Salário-base | Remuneração base de referência do cargo |
| Colab. | Quantidade de colaboradores atualmente nesse cargo |

## O que dá pra fazer aqui

### Buscar um cargo

Digite no campo **Buscar cargo…** para filtrar a lista pelo nome.

### Cadastrar um novo cargo

1. Clique em **+ Novo cargo**.
2. Preencha **Nome do cargo** (obrigatório).
3. Informe o **Código** (opcional).
4. Escolha o **Nível** (Júnior, Pleno, Sênior, Liderança).
5. Selecione o **Departamento** ao qual o cargo pertence.
6. Informe o **Salário-base**.
7. Clique em **Salvar**. O cargo passa a aparecer na lista e fica disponível no cadastro de colaboradores.

![Novo cargo](/img/manual/erp/rh-cargos-novo.png)

### Editar um cargo

1. Na linha desejada, clique no ícone de lápis (**✎**).
2. Ajuste os campos.
3. Clique em **Salvar**.

### Remover um cargo

1. Na linha desejada, clique no ícone **⨯**.
2. Confirme a mensagem **"Remover cargo?"** clicando em **Remover**.

## Dicas e observações

- O campo **Departamento** puxa os departamentos já cadastrados. Se o cargo que você quer criar pertence a um departamento que ainda não existe, cadastre-o antes em [Departamentos](./rh-departamentos.md).
- A coluna **Colab.** ajuda a entender quantas pessoas seriam afetadas antes de alterar ou remover um cargo.
