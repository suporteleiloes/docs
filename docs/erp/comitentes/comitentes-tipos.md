# Tipos de comitente

Os tipos de comitente são uma classificação livre que você cria para organizar sua carteira (por exemplo: "Banco", "Financeira", "Particular", "Órgão público"). Cada comitente pode receber um tipo, o que ajuda a filtrar e organizar a base.

## Como acessar

**ERP** → **Comitentes** → **Tipos de comitente**.

![Tipos de comitente](/img/manual/erp/comitentes-tipos.png)

## O que você vê nesta tela

Uma lista simples com os tipos já cadastrados:

| Coluna | O que é |
|---|---|
| ID | Número interno do tipo |
| Nome | Nome do tipo de comitente |
| Situação | Ativo ou Inativo |

## O que dá pra fazer aqui

### Cadastrar um novo tipo

1. Clique em **Novo tipo**.
2. Preencha o campo **Nome** (obrigatório).
3. Deixe a chave **Ativo** ligada para que o tipo fique disponível para uso.
4. Clique em **Salvar**.

![Novo tipo de comitente](/img/manual/erp/comitentes-tipos-novo.png)

### Editar um tipo

1. Clique no ícone de **lápis** (Editar) na linha desejada.
2. Altere o **Nome** ou a situação **Ativo**.
3. Clique em **Salvar**.

### Excluir um tipo

1. Clique no ícone de **lixeira** (Excluir) na linha desejada.
2. Confirme a exclusão.

## Dicas e observações

- O tipo tem apenas **Nome** e **Situação** — não há campo de descrição.
- Cada comitente recebe **um** tipo (definido no campo **Tipo de comitente** da aba Identificação do formulário).
- **Excluir é uma remoção lógica:** o tipo deixa de aparecer na lista, mas se quiser apenas pará-lo de uso preservando o cadastro, prefira **desativá-lo** (chave Ativo desligada) em vez de excluir.
- Não confunda com o filtro **Tipo de pessoa** (PF/PJ) da lista de comitentes — aquele é fixo do sistema, este aqui é a sua classificação livre.
