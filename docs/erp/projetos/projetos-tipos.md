# Tipos de Projeto

Os **tipos de projeto** são categorias que você usa para classificar seus projetos (por exemplo: Jurídico, Comercial, Operacional). Todo projeto precisa de um tipo, então mantenha essa lista organizada antes de cadastrar projetos.

## Como acessar

**ERP** → **Projetos** → botão **Tipos** (no cabeçalho da lista de projetos).

![Tipos de Projeto](/img/manual/erp/projetos-tipos.png)

## O que você vê nesta tela

Uma tabela simples com os tipos já cadastrados.

| Coluna | O que é |
|---|---|
| # | Número (ID) do tipo. |
| Nome | Nome do tipo de projeto. |
| Descrição | Texto opcional explicando o tipo. |
| Ações | Botões de editar e excluir em cada linha. |

## O que dá pra fazer aqui

- **Cadastrar**, **editar** e **excluir** tipos de projeto.
- **Voltar** para a lista de projetos pelo botão **Voltar**.

### Cadastrar um novo tipo

1. Clique em **Novo tipo**.
2. Preencha o **Nome** (obrigatório, ex.: Desenvolvimento).
3. Escreva uma **Descrição** (opcional).
4. Clique em **Criar**. O tipo passa a aparecer na lista e fica disponível ao cadastrar projetos.

![Novo tipo](/img/manual/erp/projetos-tipos-novo.png)

### Editar um tipo

1. Clique no ícone de **lápis (Editar)** na linha do tipo.
2. Ajuste o nome ou a descrição.
3. Clique em **Salvar**.

### Excluir um tipo

1. Clique no ícone de **lixeira (Excluir)** na linha.
2. Confirme na janela **Excluir tipo?**.
3. O tipo é removido.

## Dicas e observações

- O **Nome** é obrigatório — sem ele o botão de salvar fica desabilitado.
- A exclusão pede confirmação. Evite excluir tipos já usados por projetos existentes, para não deixá-los sem categoria.
