# Filas de Tarefas (globais)

As **filas globais** organizam as tarefas e a ordem de atendimento, e ficam disponíveis em **todos** os projetos. Use-as para padronizar etapas de trabalho como "Backlog", "Em revisão" ou "Concluído". Filas que servem apenas a um projeto específico são criadas dentro do próprio projeto, não aqui.

## Como acessar

**ERP** → **Projetos** → botão **Filas** (no cabeçalho da lista de projetos).

![Filas de tarefas globais](/img/manual/erp/projetos-filas.png)

## O que você vê nesta tela

Uma tabela com as filas globais cadastradas.

| Coluna | O que é |
|---|---|
| # | Número (ID) da fila. |
| Fila | Nome da fila. |
| Prioridade | Nível: Normal, Média, Alta ou Crítica. |
| Visibilidade | **Pública** (todos veem) ou **Privada** (só gestores). |
| Tarefas | Quantidade de tarefas vinculadas, quando disponível. |
| Ações | Botões de editar e excluir. |

## O que dá pra fazer aqui

- **Cadastrar**, **editar** e **excluir** filas globais.
- Definir a **prioridade** e a **visibilidade** de cada fila.
- **Voltar** para a lista de projetos.

### Cadastrar uma nova fila

1. Clique em **Nova fila**.
2. Preencha o **Nome da fila** (obrigatório, ex.: Backlog, Em revisão).
3. Escolha a **Prioridade**: Normal, Média, Alta ou Crítica.
4. Marque **Fila privada** se ela deve ficar visível só para gestores.
5. Clique em **Criar**.

![Nova fila](/img/manual/erp/projetos-filas-nova.png)

### Editar uma fila

1. Clique no ícone de **lápis (Editar)** na linha.
2. Ajuste nome, prioridade ou visibilidade.
3. Clique em **Salvar**.

### Excluir uma fila

1. Clique no ícone de **lixeira (Excluir)**.
2. Confirme na janela **Excluir fila?**.
3. A fila é removida.

## Dicas e observações

- O **Nome** é obrigatório para salvar.
- Não é possível remover uma fila que ainda tem tarefas vinculadas — o sistema avisa nesse caso. Mova ou conclua as tarefas antes de excluir.
- Estas filas são **globais**: aparecem em todos os projetos. Para uma fila exclusiva de um projeto, crie-a na aba **Filas** dentro do projeto.
