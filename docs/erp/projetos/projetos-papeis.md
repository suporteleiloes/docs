# Papéis (Grupos de Membros)

Os **papéis** definem o conjunto de permissões que um membro recebe dentro de um projeto — por exemplo, **Líder**, **Colaborador** ou **Observador**. Em vez de configurar permissão por permissão para cada pessoa, você cria papéis prontos e atribui um deles a cada membro na tela do projeto.

## Como acessar

**ERP** → **Projetos** → botão **Papéis** (no cabeçalho da lista de projetos).

![Papéis de projeto](/img/manual/erp/projetos-papeis.png)

## O que você vê nesta tela

Uma tabela com os papéis cadastrados.

| Coluna | O que é |
|---|---|
| # | Número (ID) do papel. |
| Papel | Nome do papel. |
| Descrição | Texto opcional explicando o papel. |
| Permissões | Quantidade de permissões marcadas naquele papel. |
| Ações | Botões de editar e excluir. |

## O que dá pra fazer aqui

- **Cadastrar**, **editar** e **excluir** papéis.
- Em cada papel, marcar exatamente quais **permissões** ele concede, organizadas em dois grupos: **Tarefas** e **Projeto / Membros**.
- **Voltar** para a lista de projetos.

### Cadastrar um novo papel

1. Clique em **Novo papel**.
2. Preencha o **Nome do papel** (obrigatório, ex.: Líder, Colaborador, Observador).
3. Escreva uma **Descrição** (opcional).
4. Em **Permissões**, marque as caixas das ações que o papel deve permitir.
   - Use o link **alternar todas** no cabeçalho de um grupo (Tarefas ou Projeto / Membros) para marcar/desmarcar tudo daquele grupo de uma vez.
5. Clique em **Criar**.

![Novo papel](/img/manual/erp/projetos-papeis-novo.png)

### Editar um papel

1. Clique no ícone de **lápis (Editar)** na linha.
2. Ajuste nome, descrição e as permissões marcadas.
3. Clique em **Salvar**.

### Excluir um papel

1. Clique no ícone de **lixeira (Excluir)**.
2. Confirme na janela **Excluir papel?**.
3. O papel é removido.

## Dicas e observações

- O **Nome** é obrigatório para salvar.
- As permissões disponíveis vêm do sistema (catálogo de **Tarefas** e **Projeto / Membros**). Se o catálogo não carregar, a mensagem "Catálogo de permissões indisponível" aparece — tente recarregar a página.
- Os papéis criados aqui ficam disponíveis no seletor **Papel** ao adicionar um membro na tela do projeto.
