# Validade de Documentos · Tipos

Esta tela cadastra os **Tipos** usados para classificar os documentos em [Validade de Documentos](./validade-documentos.md) — por exemplo, "Certidão", "Alvará", "Registro". Padronizar os tipos facilita filtrar e organizar os documentos.

## Como acessar

**Gestão** → **Validade de Documentos** → **Tipos**

![Validade de Documentos · Tipos](/img/manual/erp/validade-documentos-tipos.png)

## O que você vê nesta tela

No topo há a **busca** por tipo. Abaixo, a lista.

| Coluna | O que é |
|---|---|
| # | Número (identificador) do tipo. |
| Nome | Nome do tipo. |
| Descrição | Descrição do tipo (opcional). |
| Status | Indica se o tipo está **Ativo** ou **Inativo**. |

## O que dá pra fazer aqui

Em cada linha há os botões **Editar** (lápis) e **Excluir** (lixeira).

### Cadastrar um novo tipo

1. Clique em **Novo tipo**.
2. Preencha o **Nome** (obrigatório).
3. (Opcional) Escreva uma **Descrição**.
4. Defina o **Status** (Ativo ou Inativo).
5. Clique em **Criar**.

![Novo tipo](/img/manual/erp/validade-documentos-tipos-novo.png)

### Editar um tipo

1. Clique no ícone de **lápis**.
2. Ajuste o nome, a descrição ou o status.
3. Clique em **Salvar**.

### Excluir um tipo

1. Clique no ícone de **lixeira**.
2. Confirme a remoção. A exclusão é permanente.

## Dicas e observações

- Os tipos cadastrados aqui aparecem no campo **Tipo** ao criar ou editar um documento em Validade de Documentos.
- Para tirar um tipo de uso sem apagá-lo, edite-o e mude o **Status** para Inativo.
