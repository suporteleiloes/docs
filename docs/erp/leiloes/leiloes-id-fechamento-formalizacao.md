# Formalização (rota direta)

Esta é a mesma tela de **Formalização** do Fechamento, acessada pelo endereço completo do leilão (`/leiloes/{id}/fechamento/formalizacao`). É exatamente a aba **Formalização** descrita na página principal — abrir, gerar atas, baixar recibos de comissão e enviar a documentação aos arrematantes.

## Como acessar

**Leilões** → abra um leilão → **Fechamento** → aba **Formalização**.

(Ou pelo link direto que inclui o número do leilão no endereço.)
