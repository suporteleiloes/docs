# Fechamento

O Fechamento reúne tudo o que vem **depois** do pregão: oficializar as vendas, cobrar os arrematantes e emitir notas fiscais. É uma área com três abas, e a tela serve de "menu" que leva a cada uma delas.

## Como acessar

**Leilões** → abra um leilão → **Fechamento**.

(A URL termina em `/leiloes/{id}/fechamento`. Ao entrar, o sistema abre a aba **Formalização** por padrão.)

![Fechamento](/img/manual/erp/fechamento.png)

## O que você vê nesta tela

Uma barra com três abas no topo. Clique para alternar entre elas:

| Aba | Para que serve |
|---|---|
| **Formalização** | Gerar atas de arrematação, recibos de comissão e enviar a documentação aos arrematantes |
| **Cobrança** | Emitir boletos, enviar cobranças, confirmar pagamentos, recibos e termos |
| **Notas Fiscais** | Gerar, emitir, enviar e baixar as notas fiscais do leilão |

Abaixo das abas aparece o conteúdo da aba selecionada.

## O que dá pra fazer aqui

Esta tela apenas organiza o acesso. As ações ficam dentro de cada aba:

1. Clique em **Formalização** para oficializar as vendas. Veja [Formalização](./formalizacao.md).
2. Clique em **Cobrança** para cobrar e receber. Veja [Cobrança](./cobranca.md).
3. Clique em **Notas Fiscais** para emitir documentos fiscais. Veja [Notas Fiscais](./notas.md).

## Dicas e observações

- A ordem natural do fechamento é: **Formalização → Cobrança → Notas Fiscais**, mas você pode acessar qualquer aba a qualquer momento.
- A aba ativa fica destacada na barra.
