# Habilitados

Esta tela mostra quem está habilitado a participar do leilão. Aqui você acompanha as habilitações dos arrematantes, aprova ou reprova quem está em análise, adiciona arrematantes manualmente e gerencia as habilitações por lote (incluindo o direito de preferência). É a aba "Habilitados" de dentro de um leilão.

## Como acessar

**Leilões** → abra um leilão → aba **Habilitados**.

![Habilitados](/img/manual/erp/habilitados.png)

## O que você vê nesta tela

A tela tem dois blocos:

1. **Habilitações para o leilão** — a lista geral de arrematantes habilitados.
2. **Habilitações para Lotes e Direito de Preferência** — habilitações específicas por lote.

### Lista geral de habilitados

| Coluna | O que é |
|---|---|
| Apelido | Apelido do arrematante |
| ID | Identificador do arrematante |
| Nome | Nome e documento (CPF/CNPJ) do arrematante |
| Análise Cadastral | Eventuais pendências apontadas na análise |
| Habilitação | Data e hora da habilitação |
| Status | Aprovado, Em análise ou Não aprovado |

Abaixo dos filtros há os interruptores **Mostrar aprovados**, **Mostrar em análise** e **Mostrar reprovados**, que controlam quais situações aparecem na lista.

### Habilitações por lote / Direito de Preferência

| Coluna | O que é |
|---|---|
| Lote | Número do lote |
| Direito de Preferência | Solicitado, Aprovado ou Reprovado |
| Apelido / Nome | Identificação do arrematante |
| Data de habilitação | Quando foi solicitada |
| Situação | Aprovado, Em análise ou Bloqueado |

## O que dá pra fazer aqui

### Adicionar um arrematante manualmente

1. No campo **Adicionar arrematante**, comece a digitar o nome ou apelido.
2. Selecione o arrematante na lista que aparece.
3. Confirme. Ele entra como habilitado para este leilão.

### Aprovar ou reprovar uma habilitação

1. Para um arrematante **Em análise**, clique em **Aprovar** ou **Reprovar** na linha.
2. Confirme a ação. O status muda na hora.

Um arrematante já aprovado também pode ser reprovado pelo botão **Reprovar**.

### Baixar o termo de habilitação

- Clique no ícone de documento na linha do arrematante para baixar o termo digital em PDF.

### Excluir uma habilitação

- Clique em **Excluir** na linha e confirme.

### Exportar a base de habilitados

- Clique em **Exportar** para baixar a lista atual em planilha (XLSX).
- Clique em **Exportar tudo** para baixar a base completa.

### Gerenciar habilitações por lote (Direito de Preferência)

1. Use o campo **Filtrar por nº de lote** para localizar um lote.
2. Para uma habilitação em análise, clique em **Aprovar**, **Aprovar c/ DP** (aprova com direito de preferência) ou **Reprovar**.
3. Use **Excluir** para remover o registro.

## Dicas e observações

- Se o leilão exige análise obrigatória das habilitações, use o link para o **Painel de Análise de Habilitações** (no aviso no topo da tela) para acompanhar as análises pendentes de forma centralizada.
- Os interruptores de exibição (aprovados/em análise/reprovados) e a configuração de colunas ficam salvos no seu navegador.
- Habilitações também podem chegar automaticamente pelo portal/site, sem precisar adicionar manualmente.
