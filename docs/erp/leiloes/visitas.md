# Visitas

A aba **Visitas** mostra, em tempo real, quantas pessoas acessaram o leilão e seus lotes. Os números vêm da mesma fonte que alimenta o site público, então o que você vê aqui é o que o público vê lá. É a tela ideal para acompanhar o interesse pelo leilão antes e durante o pregão.

## Como acessar

Abra um leilão e, no menu do leilão, clique em **Visitas**.

![Visitas](/img/manual/erp/visitas.png)

## O que você vê nesta tela

A tela tem três áreas: os indicadores no topo, a tabela de acessos por dia e o ranking dos lotes mais visitados.

### Indicadores (KPIs)

| Indicador | O que é |
|---|---|
| Total de visitas ao leilão | Soma de todos os acessos diários ao leilão. |
| Total exibido no site público | Número que aparece no site público. Em condições normais bate com o total acima; quando diverge, a tela avisa. |
| Mobile | Acessos via celular/tablet. |
| Desktop | Acessos via desktop/notebook. |

### Acessos por dia

Tabela com os últimos dias, mostrando **Data**, **Acessos**, **Mobile** e **Desktop**. Se ainda não houver registros, a tela explica que os dados são gerados quando o cache do leilão é atualizado.

### Top 5 lotes mais visitados

Ranking dos lotes que mais receberam visitas, com **Lote** (número), **Descrição** e **Visitas**. Acima do ranking, a tela mostra a soma de visitas em todos os lotes.

## O que dá pra fazer aqui

Esta tela é de consulta. A ação principal é acompanhar os números e, quando precisar de um documento, ir para o relatório:

### Baixar o PDF de visitas

1. No texto de descrição (ou na aba de Relatórios), use o relatório **Visitas por Lote**.
2. Ele gera o PDF com as visitas por lote — o mesmo número exibido nesta tela.

## Dicas e observações

- Os números vêm do **cache de acessos** do leilão (as mesmas tabelas que alimentam o site público); para download em PDF, use o relatório **Visitas por Lote**.
- Se os indicadores estiverem **zerados** ou a tabela de acessos por dia estiver vazia, é porque o cache ainda não foi gerado. Ele é atualizado quando o cache do leilão é recalculado (ao abrir o Dashboard do leilão ou em rotina periódica). Visitas só começam a ser contadas depois que o leilão está publicado no site.
- Se a tela apontar **divergência** entre o total do leilão e o total do site público, abra o Dashboard do leilão para forçar a atualização do cache. Persistindo, acione o suporte.
