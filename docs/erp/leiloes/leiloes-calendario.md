# Calendário de Leilões

Esta tela mostra seus leilões em formato de calendário mensal. É uma forma visual de enxergar a agenda: em que dias há leilões, quantos e em que situação. Ótimo para planejar a semana e abrir rapidamente um leilão a partir da data.

## Como acessar

**Leilões** → **Calendário**.

![Calendário de Leilões](/img/manual/erp/leiloes-calendario.png)

## O que você vê nesta tela

### Barra de navegação e filtros

No topo há os botões para avançar e voltar de mês (setas), o nome do mês corrente e a contagem de leilões do mês. O botão **Hoje** (no cabeçalho) volta para o mês atual.

À direita ficam os filtros: **Status**, **Leiloeiro** e **Comitente**. Quando algum estiver aplicado, aparece o botão **Limpar**.

### Grade do mês

A grade mostra os dias do mês, com os finais de semana destacados e o dia de hoje marcado. Em cada dia com leilões aparece:

- um contador com a quantidade de leilões no dia;
- um cartão por leilão, com o **horário**, o **título** e o **status**.

Cada cartão é colorido conforme o status do leilão. Abaixo da grade há uma **legenda** explicando as cores:

| Cor | Significado |
|---|---|
| Aberto | Leilão aberto para lances |
| Em leilão / Adiado / Suspenso | Leilão em andamento ou em situação de pausa |
| Encerrado | Leilão finalizado |
| Cancelado | Leilão cancelado |
| Rascunho / Em breve | Leilão ainda não publicado / agendado |

## O que dá pra fazer aqui

### Navegar pela agenda

1. Use as setas para mudar de mês.
2. Clique em **Hoje** para voltar ao mês atual.

### Filtrar os leilões exibidos

1. Escolha um **Status**, um **Leiloeiro** e/ou um **Comitente** nos seletores.
2. Para remover os filtros, clique em **Limpar**.

### Abrir um leilão a partir do calendário

- Clique no cartão do leilão dentro do dia. Você é levado direto para a página daquele leilão.

### Criar um leilão

- Clique em **Novo leilão** (no cabeçalho) para abrir o formulário de cadastro.

## Dicas e observações

- O número que aparece no canto de cada dia é a quantidade de leilões agendados naquele dia.
- A posição do leilão no calendário segue a data de agenda do leilão (próximo leilão).
