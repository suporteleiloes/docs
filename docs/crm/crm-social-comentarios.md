# Comentários (moderação)

Esta tela reúne os comentários públicos das suas redes sociais para você moderar. Cada comentário já vem classificado por sentimento (positivo, neutro ou negativo) por inteligência artificial, o que ajuda a priorizar críticas e ocultar conteúdo tóxico.

## Como acessar
**CRM** → **Social** → **Comentários**.

![Comentários (moderação)](/img/manual/crm/crm-social-comentarios.png)

## O que você vê nesta tela

No topo há filtros por sentimento (**Todos**, **Positivos**, **Neutros**, **Negativos**) e um botão **Mostrar ocultos** para incluir os comentários já ocultados. Abaixo, a lista de comentários.

### Cada comentário mostra

| Elemento | O que é |
|---|---|
| **Autor** | Nome de quem comentou (ou "Anônimo") e a rede de origem (Instagram, Facebook, LinkedIn, Twitter). |
| **Sentimento** | Etiqueta colorida com a classificação da IA (positivo, neutro, negativo) e a pontuação entre parênteses. |
| **Corpo** | O texto do comentário. |
| **Data** | Há quanto tempo o comentário foi feito (ex.: "15min atrás", "3h atrás"). |
| **Ocultar / Oculto** | Botão para ocultar; se já estiver oculto, mostra a etiqueta "Oculto". |

## O que dá pra fazer aqui

### Filtrar comentários
1. Clique em um dos chips de sentimento (**Positivos**, **Neutros**, **Negativos**) para ver apenas aquele grupo. Clique em **Todos** para limpar o filtro.

### Mostrar comentários ocultos
1. Ative o botão **Mostrar ocultos** para incluir na lista os comentários que já foram ocultados (aparecem esmaecidos com a etiqueta "Oculto").

### Ocultar um comentário
1. Localize o comentário (por exemplo, um negativo ou ofensivo).
2. Clique em **Ocultar**.
3. Confirme. O comentário fica marcado como oculto.

## Dicas e observações

- O **sentimento** é sugerido por IA. Use como guia, mas revise: a IA pode errar em ironia ou gírias.
- **Responder diretamente** o comentário na rede externa depende de a integração com a Meta estar ativa.
- Comentários só aparecem aqui depois que chegam pelos webhooks das integrações configuradas. Se a lista estiver vazia, confira se as integrações estão ativas.
