# Minhas entregas

Esta é a visão **pessoal e somente leitura** das entregas ligadas à sua própria conta. Diferente da tela de [Entregas](./entrega-itens.md) (usada pela operação para gerenciar tudo), aqui você vê apenas as suas entregas e o status de cada uma — sem ações administrativas.

## Como acessar

**Conta** → **Minhas entregas** (rota `/minhas-entregas`).

![Minhas entregas](/img/manual/crm/minhas-entregas.png)

## O que você vê nesta tela

Uma lista de cartões, um por entrega. Cada cartão traz:

| Elemento | O que é |
|---|---|
| **Código** | Identificador da entrega. |
| Situação | Status atual (ex.: Disponível p/ retirada, Entregue). |
| Etiqueta **Atrasada** | Aparece quando passou da data prevista. |
| **Pronta para retirada** | Aviso em destaque quando o item já pode ser retirado. |
| Itens | Lista dos itens da entrega (tipo e descrição). |
| **Como retirar** | Orientações de retirada, quando preenchidas pela operação. |
| Leilão / Lote | O leilão e o lote relacionados. |
| Prevista | Data prevista da entrega. |

## O que dá pra fazer aqui

Esta tela é **de consulta**. Use-a para:

1. Acompanhar o **status** de cada entrega.
2. Ver quando uma entrega fica **pronta para retirada**.
3. Ler as **orientações de retirada** (onde e como retirar).
4. Conferir os **itens** e a **data prevista**.

Se você não tiver entregas, a tela mostra "Você não tem entregas no momento".

## Dicas e observações

- A lista mostra **apenas as entregas da sua conta** — você não vê dados de outras pessoas. O sistema identifica suas entregas pela pessoa vinculada ao seu usuário; se o seu login não estiver vinculado a um cadastro de pessoa, a tela aparece vazia mesmo que existam entregas no seu nome (peça à operação para vincular).
- **Entregas canceladas não aparecem** nesta lista.
- A etiqueta **Pronta para retirada** aparece quando a entrega está em **Disponível p/ retirada** ou **Enviado**.
- Não há botões de edição aqui. Para criar, alterar ou confirmar entregas, a operação usa a tela de [Entregas](./entrega-itens.md).
- Quando aparecer "Pronta para retirada", siga as instruções do campo **Como retirar** para buscar o item.
