# Menus

Esta tela gerencia os **itens de menu** do site — os links que aparecem no menu principal (topo) e no rodapé (footer) do site público. Cada item tem um título, um link de destino e a seção onde aparece.

## Como acessar

**Sites** → **Menus**

![Menus](/img/manual/sites/website-menus.png)

## O que você vê nesta tela

Um campo de busca no topo e a tabela com os itens de menu.

| Coluna | O que é |
|---|---|
| # | Número de identificação do item. |
| Título | Texto que aparece no menu. |
| Link | Endereço para onde o item leva. |
| Seção | **Principal** (menu do topo) ou **Footer** (rodapé). |

Use o campo **Buscar menu...** para filtrar.

## O que dá pra fazer aqui

### Cadastrar um novo item de menu

1. Clique em **Novo menu** no topo.
2. Preencha o **Título** (obrigatório) — texto do link, ex.: "Sobre nós".
3. Preencha o **Link** (obrigatório) — destino, ex.: `https://dominio.com.br/page`.
4. Em **Seção**, escolha **Principal** (menu do topo) ou **Footer** (rodapé).
5. Clique em **Cadastrar**.

![Cadastro de item de menu](/img/manual/sites/website-menus-novo.png)

### Editar um item de menu

1. Dê duplo clique na linha ou clique no ícone de **lápis**.
2. Ajuste os campos e clique em **Atualizar**.

### Excluir um item de menu

1. Clique no ícone de **lixeira** na linha.
2. Confirme. **A exclusão é permanente.**

## Dicas e observações

- **Título, Link e Seção** são todos obrigatórios — o botão **Cadastrar/Atualizar** só habilita quando os três estão preenchidos.
- Há apenas duas seções: **Principal** (topo) e **Footer** (rodapé).
- Para apontar um item de menu para uma página criada no site, use a URL dessa página (veja [Páginas](./website-paginas.md)).
